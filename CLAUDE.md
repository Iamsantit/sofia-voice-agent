# Agente de Voz Sofia - Inmobiliaria

## Descripción

Sistema de inteligencia artificial con agente de voz para inmobiliaria. Sofia es un agente capaz de recibir llamadas, hacer llamadas salientes y gestionar mensajes, todo con contexto inmobiliario (propiedades, citas, seguimiento de leads).

## Stack

- **Retell AI** — Plataforma de agentes de voz (orquestación de conversaciones, LLM, voz)
- **Modal** — Hosting de automatizaciones y funciones serverless (webhooks, lógica de negocio)
- **Twilio** — Números telefónicos, enrutamiento de llamadas y SMS
- **Python** — Lenguaje principal para backend/automatizaciones

## Capacidades del Agente

1. **Recibir llamadas** — Atender llamadas entrantes de clientes/leads
2. **Enviar llamadas** — Llamadas salientes de seguimiento y confirmación
3. **Recibir mensajes** — Procesar SMS/mensajes entrantes

## Arquitectura (alto nivel)

```
Twilio (número) → Retell AI (agente de voz) → Modal (lógica de negocio/webhooks)
```

- Twilio gestiona la telefonía (números, enrutamiento)
- Retell AI maneja la conversación de voz (STT, LLM, TTS)
- Modal ejecuta la lógica de negocio (webhooks, integraciones, automatizaciones)

## Convenciones

- Código y variables en inglés
- Comentarios y documentación en español cuando sea necesario
- Funciones de Modal con decoradores estándar (`@app.function`, `@web_endpoint`)
- Secrets y API keys en variables de entorno, nunca hardcodeados
